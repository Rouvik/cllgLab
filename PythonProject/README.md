# How to setup and use?

## First create a python virtual environment:
Run `python -m venv env`

## Then activate the environment:
From the base project folder where `env/` folder is present.

- **Windows**:

  Run to activate:
  + `./env/scripts/activate.ps1` or
  + `./env/scripts/activate.bat` or
  + whatever is close to activate
  
  and `deactivate` to deactivate environment.

- **Linux**:

  Run `source ./env/bin/activate` to activate and `deactivate` to deactivate environment.

## Once in environment, get packages:
Run `pip install -r requirements.txt`

## Now run the programs!
Run `python ./fromProject/Main.py` for group project.

Run `python ./Calculator/calculator.py` for individual calculator project.
